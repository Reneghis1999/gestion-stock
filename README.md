# Gestionnaire de Stock (v1 – Python Procédural)

## Description
Ce projet est un mini-système de gestion de stock développé en **Python procédural**.  
Il permet de gérer des produits en console : ajouter, supprimer, modifier, rechercher et afficher le stock.  

Cette version a pour objectif de **consolider les bases Python** :  
- Variables et types  
- Listes et dictionnaires  
- Boucles et conditions  
- Fonctions avec paramètres et valeurs de retour  
- Interaction avec l’utilisateur (`input` / `print`)  

---

## Fonctionnalités

- Ajouter un produit au stock  
- Supprimer un produit du stock  
- Modifier les informations d’un produit (quantité, prix)  
- Rechercher un produit par nom  
- Afficher l’ensemble du stock
